How to use Time2eat

home
View recipes

On the home page you can view all the recipes. They are sorted by the number of votes and number of views, with the most voted being placed on the top of the carousel of images. They are sorted on descending order if you move to the right, and on ascending order if you move to left.

Below the carousel you can find the filter section where you can filter the recipes based on different criteria (i.e. you could filter only by cuisine or you could filter by cuisine, type of meal, allergens to exclude, and difficulty).

On the top right hand side there is a search input box. You can use this input box to look for certain ingredients, authors, name of recipes, allergens, levels of difficulty, cuisines or types of meals.

trending_up
Recipes dashboard

In this section you can have an overview of the recipes included in the app. You can see the number of recipes split by different criteria and also a summary table with some more details.

Also in this section you can see the top 10 rated authors.

settings
Manage type of meal / cuisines

In this section you can see all the type of meals and cuisines that already exist in the app.

If you are planning to add recipes and the type of meal or cuisine is not included in the list make sure you add it before starting adding a new recipe.

add_box
Add recipe

Use this section to add your recipes.

* Recipes with pictures generally receive more votes so we recommend adding an image. If you don't upload an image we will add a standard picture to your recipe.
